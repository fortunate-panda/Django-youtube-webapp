# PyTube - Django Video Sharing App

PyTube is a lightweight, YouTube-like web application built with Python and Django. It allows users to upload, list, and watch video files using standard HTML5 video playback.

## Features
* **Video Uploading:** Secure file uploading for video content.
* **Video Feed:** A responsive dashboard listing all uploaded videos.
* **Video Playback:** Dedicated watch pages utilizing the HTML5 `<video>` tag.
* **Responsive UI:** Styled using Bootstrap 5 for mobile and desktop viewing.

## Prerequisites
* Python 3.8+
* pip (Python package installer)
